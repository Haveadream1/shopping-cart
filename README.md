# Shopping Cart

TOP project of a 3 pages website, composed of a home page, shop page and cart page. Allow to browse a selection of items retrieved with API call.

## Goals

Practice website with React, allow the use for the first time of router and DataContext   
As well as introduction to Vitest with module test for components
